Printer Plugin Demo

The Printer plugin allows you to wirelessly print a screenshot of your app. On iOS you can even print any DOM object on the page.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android - App package.

Can we print?

Any modern iOS device is capable of using AirPrint to find connected printers. On Android though the app needs to run on 4.4 (KitKat) or higher.
